＜使用方法＞
・必要なもの
Rubyが実行可能な環境。（開発環境：Windows、ruby 3.3.7）

・クローンして実行
git clone https://github.com/gr9pe/wordle-like-game.git
cd path\to\this\app\bin
ruby index.rb

・操作方法
1．難易度を1～3の間で選択してください。
　1...簡単。40語の中からランダムで出題。
　2...普通。約300語の中からランダムで出題。
　3...かなり難しい。約25000語の中からランダムで出題。

2．キーボードから1文字入力してください。（エンター不要です）

3．Ctrl＋Cで終了します

＜拡張仕様の実装＞
・コンソール上に、これまで⼊⼒した単語の⼀覧を表⽰する
・ゲーム終了時に、正解の単語を表⽰する
・ゲーム終了時に、もう⼀度実⾏するかをユーザーに訪ねるようにし、連続でゲームが遊べるようにする
・連続でゲームを遊ぶとき、ランダムに選ばれる単語が以前と被らないようにする
・難易度を設定できるようにする。例えばイージー‧ノーマル‧ハードを選択できるようにし、それぞれ失敗可能回数や単語⽂字数が異なるようにする
