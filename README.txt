＜使用方法＞
・必要なもの
Rubyが実行可能な環境。

・実行コマンド
path\to\this\app\$ ruby index.rb

・操作方法
キーボードから1文字入力してください。（エンター不要です）


＜どのように作ったか＞
・言語の選定

・規模感

・調べたこと

3．型について（型の扱いや標準の関数がJavaと異なるため）
4. フィールドの巻き取りとスコープ（Javaと異なるため）
